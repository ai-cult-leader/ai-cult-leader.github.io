What is PostHog?

PostHog is the open-source product analytics suite combining analytics, session recording, feature flags, A/B testing, and surveys in one platform. Engineers prefer it because they can self-host for data privacy, contribute to open source, and get all tools in one platform instead of paying for multiple separate subscriptions.
