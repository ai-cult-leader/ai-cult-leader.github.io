📦 Setup Instructions
1. Go to Discord Developer Portal → New Application → Bot tab
2. Copy your bot token and replace YOUR_BOT_TOKEN
3. Enable Message Content Intent in the Bot tab
4. Install: pip install discord.py
5. Invite bot with OAuth2 URL (scopes: bot, slash commands)
6. Run: python bot.py
