🚀 How to Install

1
Open your Google Sheet → click Extensions → Apps Script
2
Delete any existing code, then paste your generated script
3
Click Save (floppy disk icon) and name your project
4
To set up triggers: click Triggers (clock icon) → Add Trigger → configure as needed
5
First run: click Run → authorize permissions when prompted
